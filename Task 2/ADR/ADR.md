|-----|-------|--------|
| [ADR-001](adr-001.md) | Use CRDT (Yjs) Over Operational Transformation for Conflict Resolution | Accepted |
| [ADR-002](adr-002.md) | Use WebSocket for Real-Time Communication Over HTTP Polling | Accepted |
| [ADR-003](adr-003.md) | Web-Based Editor Over VS Code Extension | Accepted |
| [ADR-004](adr-004.md) | Stateless Relay Server Over Centralized Document Authority | Accepted |
